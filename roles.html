👥 INDIVIDUAL ROLES

// Tactical Responsibilities
Squad Leader (SL) ⭐
Command
  • Menentukan navigasi dan objektif regu Mondragon.
  • Komunikasi ke HQ via Long Range Radio.
  • Menandai peta (Marking) untuk intelijen.
EQ: Radio Backpack, Binoculars, Map.
Second-in-Command (2IC) 🎖️
Leadership
  • Mengambil alih komando penuh jika SL gugur.
  • Menjaga barisan belakang (Rear Security).
  • Monitor amunisi dan stamina anggota regu.
SOP: Memimpin elemen manuver (Fireteam).
Forward Observer (FO) 📡
Asset Control
  • Menghitung koordinat presisi target musuh.
  • Meminta dan memandu serangan artileri/udara.
  • Membatalkan tembakan jika terlalu dekat kawan (Danger Close).
EQ: Vector Rangefinder, Laser Designator.
Combat Medic 🏥
Medical
  • Melakukan Triage (menentukan prioritas pasien).
  • Menghentikan pendarahan dan mengembalikan tensi darah (Saline).
  • Menarik rekan (Drag/Carry) ke area aman sebelum ditolong.
SOP: Keselamatan Medic di atas segalanya.
Light Anti-Tank (LAT) 🚀
Anti-Armor
  • Eliminasi kendaraan lapis baja ringan (BTR, APC).
  • Menghancurkan gedung yang diduduki musuh.
  • Selalu peringatkan kawan dari *Backblast* sebelum menembak.
EQ: Disposable Launcher (LAW, RPG-26).
Heavy Anti-Tank (HAT) 💥
Armor Annihilation
  • Menghancurkan Main Battle Tank (MBT).
  • Menembak dari jarak jauh menggunakan misil berpemandu.
  • Membutuhkan Asisten untuk memuat ulang roket.
EQ: Javelin, Kornet, atau Carl Gustaf.
Recon Scout 👁️
Infiltration
  • Membuka jalur aman untuk Peleton Utama.
  • Mengumpulkan intelijen musuh tanpa terdeteksi.
  • Penyergapan senyap.
EQ: Silenced Weapon, Ghillie Suit, Binoc.
Sniper 🎯
Precision Lethality
  • Eliminasi HVT (Perwira musuh, Sniper musuh, Gunner statis).
  • Bekerja berdasarkan kalkulasi dari Spotter.
  • Memberikan perlindungan *Overwatch* untuk regu infanteri.
EQ: Bolt-Action/Semi-Auto Sniper Rifle.
Spotter 🔭
Target Calculation
  • Mencari dan mengidentifikasi target untuk Sniper.
  • Menghitung jarak, arah angin (*windage*), dan elevasi.
  • Melindungi posisi Sniper dari jarak dekat (CQB Security).
EQ: Rangefinder, Spotting Scope, Assault Rifle.
Automatic Rifleman (MG) 🔫
Firepower
  • Memberikan tembakan *Suppression* yang akurat.
  • Memegang kendali pertempuran (*Base of Fire*).
  • Menghemat stamina agar siap posisi *bipod* kapan saja.
EQ: Light/Medium Machine Gun.
Ammo Bearer 📦
Resupply
  • Bertindak sebagai *Battle Buddy* bagi Machine Gunner/HAT.
  • Membawa suplai amunisi berat yang tidak bisa dibawa penembak.
  • Siap mengambil alih senjata utama jika Gunner gugur.
EQ: Heavy Backpack, Extra Ammo/Rockets.
Grenadier / Rifleman 💣
Assault
  • Ujung tombak pertempuran jarak dekat.
  • Menggunakan granat asap untuk penyamaran regu.
  • Membuka rute mati menggunakan pelontar granat (HE).
EQ: Underbarrel Launcher (UGL), Smoke.
Mortar Gunner 💣
Indirect Fire
  • Mengeset elevasi dan azimuth tabung mortir.
  • Menembakkan proyektil High Explosive atau Smoke.
  • Berkomunikasi secara terus menerus dengan FO untuk koreksi tembakan.
SOP: Wajib paham kalkulasi artileri dasar.
Combat Engineer 🛠️
Infrastructure
  • Membangun Forward Operating Base (FOB).
  • Membangun benteng pertahanan (HESCO, Sandbags).
  • Mendeteksi dan menjinakkan ranjau (EOD).
EQ: Entrenching Tool, Mine Detector.
Logistics Driver 🚛
Supply Chain
  • Mengemudikan truk pasokan dari Main Base ke Frontline.
  • Mengelola *Supply Points* agar regu bisa *respawn/rearm*.
  • Melakukan perbaikan darurat pada kendaraan tempur.
SOP: Menghindari rute utama yang berpotensi *ambush*.
